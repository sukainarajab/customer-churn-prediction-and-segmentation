# Customer Churn Prediction and Segmentation

## Overview
This project predicts customer churn using Machine Learning and segments customers into meaningful groups using K-Means clustering.

## Features
- Data preprocessing
- Exploratory Data Analysis (EDA)
- Random Forest Classification
- Customer Segmentation using K-Means
- Data Visualization and Insights

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Models Used
- Random Forest Classifier
- K-Means Clustering

## Results
- Model Accuracy: ~79%
- Customer Segments:
  - High Risk Customers
  - Loyal Premium Customers
  - New Regular Customers
  - Stable Long-Term Customers
